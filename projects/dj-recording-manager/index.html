DJ Recording Manager

Automated DJ Set Recording, Scheduling & File Management
Full-stack Web App  |  Node.js, React, OBS Studio

A full-stack web application for managing and automating DJ set recordings using OBS Studio. Features include remote recording control, schedule automation, real-time status dashboard, file management, and REST API endpoints.

  • OBS integration (start/stop, monitor, manage recordings)
  • Schedule management with conflict prevention
  • Dynamic directory creation and file naming
  • Real-time UI and status dashboard
  • API endpoints for automation

Tech Stack: React, Vite, Tailwind CSS, Node.js, Express, obs-websocket-js, OBS Studio
