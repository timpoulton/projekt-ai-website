ManyChat Guestlist Automation

Automated guest list, ticketing & messaging system leveraging ManyChat and Google Sheets

Services

  • Chatbot Automation
  • API Integration
  • Guestlist Management
  • Data Sync & Analytics
  • Automated Messaging
Overview

Challenge

Managing nightly guest lists, ticketing, and last-minute changes was entirely manual, error-prone, and slow. Staff juggled spreadsheets, SMS threads, and club management software—all while doors were opening.

Approach

We designed a ManyChat-driven workflow that pulls event data from the booking API, collects guest details in Messenger, synchronises everything to Google Sheets, and issues dynamic QR tickets via Twilio—all without human intervention.

Process
01

Discovery & Requirements

Interviewed venue staff, mapped existing processes, and defined automation success metrics.

02

Chatbot Architecture

Built a modular ManyChat flow handling guest sign-up, ticket issuance, and status updates.

03

API Integration

Connected booking software and Google Sheets via Apps Script, ensuring real-time data sync.

04

Testing & Launch

Executed test runs during off-peak nights, iterated on feedback, and rolled out to live events.

Patron Rating System
